Diversify-project

Experiments for emerging software diversity

The DIVERSIFY-project organization

This Github organization hosts a set of prototypes that aim at investigating the engineering and emergence of software diversity in collaborative adaptive systems. The work of this organization is supported by the FP7 STREP DIVERSIFY project
